# Pokemon typing version 1.28

Mobile layout fixes for Type Effectiveness and Stats pages.

## Changes

- Type Effectiveness mobile header now participates in normal page flow instead of reserving a large invisible spacer.
- Removed mobile-only 500px/900px body padding that caused cards to start far down the page.
- Stats mobile header now starts at the top instead of midway down the screen.
- Stats results now sit directly below the mobile header and use a single-column layout.
- Desktop layouts are unchanged.
